# AI Delver

Welcome to the documentation for **AI Delver**, an agentic AI reinforcement learning framework that trains agents to solve level trajectories.

## Overview
This documentation provides configuration guidelines, design documentation, and execution references for the project modules.

### Getting Started
Use the navigation sidebar to explore documentation for specific utilities like the command-line training client.
